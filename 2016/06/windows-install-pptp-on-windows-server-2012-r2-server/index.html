[Windows] 在 Windows Server 2012 R2 上安裝 PPTP Server

發表於 2016-06-11 | 分類於 Windows |

有時候需要VPN來遠端使用一些本地服務，或是需要跳板IP的時候，就需要VPN這個服務啦，先前有整理過Linux的VPN安裝教學，這次來整理一下Windows的囉！

 

安裝腳色

  1. 將遠端存取打勾
  2. 無須異動直接下一步
  3. 下一步
  4. 將DirectAccess與VPN(RAS)打勾
  5. 會挑出視窗詢問是否安裝下列功能
  6. 將路由打勾
  7. 點擊安裝
  8. 安裝過程

     

新增VPN

  1. 在伺服器管理員裡的工具選取遠端存取工具
  2. 點擊左上角的DirectAccess與VPN
  3. 點擊右邊的開啟RRAS管理
  4. 選擇設定和啟用路由及遠端存取
  5. 下一步
  6. 選取自訂設定
  7. 將VPN存取與NAT打勾
  8. 完成
  9. 新增完成當然要啟動服務囉！

     

設定VPN與NAT

  1. 選擇內容
  2. 上面選項點擊IPv4並點選靜態位置集區
  3. 設定IP範圍
  4. 設定完後畫面如下，點擊確定
  5. 展開IPv4找到NAT點擊右鍵選取新增介面
  6. 選擇網卡

  7. 選擇連線到網際網路的公用介面並將在這個介面上啟用NAT打勾

     

    新增VPN使用者

  8. 在電腦管理的使用者裡面點選右鍵點擊新使用者

  9. 輸入使用者名稱與密碼，並將使用者不能變更密碼與密碼永久有效打勾
  10. 對剛剛新增的使用者點擊右鍵按內容
  11. 在上方找到撥入並將網路存取使用權限設定為允許存取

     

設定完成後就可以使用裝置進行PPTP連線囉！
如果您的Server前面還有一層路由的話記得開啟Port轉發，以免無法連線！

For PPTP: 1723 TCP and Protocol 47 GRE (also known as PPTP Pass-through
For L2TP over IPSEC: 1701 TCP and 500 UDP
For SSTP: 443 TCP
